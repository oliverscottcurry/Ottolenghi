# Random Ottolenghi Salad Maker

This is a simple HTML & JavaScript web application that generates a random Ottolenghi-style salad.

## Features
- Generates a unique salad each time you click the button.
- Uses a variety of ingredients from different categories (greens, grains, vegetables, proteins, dressings, and wildcards).
- Stylish, modern UI with clear ingredient and method sections.

## Installation
1. Clone or download this repository.
2. Open `salad.html` in your web browser.
3. Access this link: [Ottolenghi Salad Generator](https://oliverscottcurry.github.io/ottolenghi/salad.html)

## Usage
1. Open the `salad.html` file in any modern web browser.
2. Click the **Generate Salad** button to create a random salad recipe.
3. Enjoy the unique combination of flavours!

## Customisation
- To add more ingredients, edit the JavaScript arrays in `salad.html`.
- Modify the CSS in the `<style>` section to change the look and feel.
